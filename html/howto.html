How to Play

Your character is represented by the '@' sign. You move it with the arrow keys.

Hovering with the mouse on the map you get info about a tile.

(The game is very much a work in progress, there is not much to do yet, except entering/exiting sites).

Other Commands

  • G: get item (from the floor)
  • V: view information about a tile
  • I: show inventory
  • S: save game
  • L: load game
